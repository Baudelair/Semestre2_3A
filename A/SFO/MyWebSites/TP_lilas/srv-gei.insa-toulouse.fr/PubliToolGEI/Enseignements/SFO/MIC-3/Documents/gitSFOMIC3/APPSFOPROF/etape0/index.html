1 Installation de Lilas en TP

Le système LilasV4 travaille à partir de deux répertoires : le répertoire d'installation (désigné par LILASHOME) /home/commetud/3eme\ Annee\ MIC/APPSFO/LILASHOME et le répertoire de travail ~/APPSFO.

Pour créer votre répertoire de travail, utilisez la commande suivante :

$ /home/commetud/3eme\ Annee\ MIC/APPSFO/LILASHOME/installAPPSFO.sh
Effectuez les tests comme indiqué
$ source ~/.bashrc $ cd ~/APPSFO $ lilas -version Lilas Version V4 du xx xx 201x xx:xx:xx

Autre test possible (avant installation) de la machine Java et Adept :

$ java -jar /home/commetud/3eme\ Annee\ MIC/APPSFO/LILASHOME/lib/lilasV4.jar -config=/home/commetud/3eme\ Annee\ MIC/APPSFO/LILASHOME/lilasConfigTestInstallation.xml etape0/TestInstallation.class -special

2 Installation de Lilas sur votre machine perso

Installez le répertoire simplement en dézippant le fichier lilasApp.zip dans un répertoire quelconque de votre espace de travail, par exemple votre répertoire principal (/home/xxx); vous obtiendrez alors le répertoire de nom LILASHOME.

Pour créer votre répertoire de travail,  :

  • Première manière : sous linux ou window, rendre exécutable le fichier LILASHOME/lib/lilasV4.jar, puis cliquez sur ce fichier pour l'exécuter.
  • Deuxième manière :
    • sous linux :
      $ ~/LILASHOME/runLILASV4.sh
    • sous window (modifier le fichier de commande pour utiliser les bons répertoires pour java, éventuellement), cliquer sur le fichier LILASHOME/runLILASV4.bat
Une fenêtre demandant la création du fichier de configuration lilasConfig.xml apparaît. Cliquer sur "Oui".

Une fenêtre demande ensuite l'enregistrement du fichier lilasConfig.xml. Créer un répertoire de travail, par exemple APPSFO, et demander l'enregistrement du fichier lilasConfig.xml dans ce répertoire.

Pour lancer lilas, utilisez simplement le fichier lilas.sh créé dans le répertoire APPSFO (sous window, créer un fichier bat analogue).
