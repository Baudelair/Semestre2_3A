BNF de l'assembleur

Start := ( ( Etiquette )? ( DirEqu | DirWord | DirOrg | Instruction | LigneVide) )* Etiquette := ( IDENTIFIER ":" ) Identificateur := ( IDENTIFIER ) Entier := ( INTEGER_LITERAL ) DirEqu := EQU INTEGER_LITERAL FinDeLigne DirWord := WORD ( Entier | Identificateur ) FinDeLigne DirOrg := ORG INTEGER_LITERAL FinDeLigne LigneVide := FinDeLigne Instruction := ( InstReg | InstBr ) FinDeLigne Immediat8 := ( Identificateur ) Immediat16 := "#" ( Entier | Identificateur ) Registre := "R" ["0"-"15"] IndirectRegistre := "[" Registre "]" IndirectImmediat16 := "[" "#" ( Entier | Identificateur ) "]" Instruction := ( Identificateur ( ( Registre | Immediat8 | Immediat16 | Identificateur) ( , ( Registre | Immediat16 | Identificateur | IndirectRegistre | IndirectImmediat16 ) )? )? ) FinDeLigne FinDeLigne := "caractère de fin de ligne" INTEGER_LITERAL := DECIMAL_LITERAL | HEX_LITERAL | BINARY_LITERAL DECIMAL_LITERAL := (["0"-"9"])+ HEX_LITERAL := "0" ["x","X"] (["0"-"9","a"-"f","A"-"F"])+ BINARY_LITERAL := "0" ["b","B"] (["0"-"1"])* IDENTIFIER := "_","a"-"z","A"-"Z" ("_","a"-"z","A"-"Z" | "0"-"9")*
Les commentaires apparaissent entre ";" et la fin de la ligne. L'identificateur d'instruction doit être l'un des codes opérations parmi ceux indiqués dans le sujet de l'étape 1.
