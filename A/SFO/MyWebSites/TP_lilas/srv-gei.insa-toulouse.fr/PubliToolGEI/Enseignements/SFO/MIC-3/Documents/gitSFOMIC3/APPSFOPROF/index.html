2 Dernières versions des logiciels installés

Si vous ne travaillez pas directement à partir du logiciel installé dans les salles de TP de l'INSA, vérifiez que vous avez les dernières versions des fichiers assembleur.jar, appsfo.jar et lilasV4.jar se trouvant dans votre répertoire d'installation LILASHOME/lib.

Utilisez pour cela la commande : lilas -version; elle doit fournir les informations suivantes :

$ lilas -version Jars Versions lilasV4demo.jar => Version V4 du 13 February 2017 10:00:46 lilasV4.jar => Version V4 du 28 February 2017 16:22:31 Lilas terminé !
