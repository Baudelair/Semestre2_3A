Assemblage d'instructions inconnues de l'assembleur

Vous avez défini l'instruction machine SWAP (COP=OxB, SCO=0x1) et vous voulez l'utiliser dans le programme ci-dessous alors que l'assembleur ne connait pas cette instruction. Que vous faut-il mettre dans la ligne de manière à ce que l'assembleur génère le code correct et validez votre réponse par le bouton 'Envoyer' :

ORG 0 WORD 0x400 ; initialisation de SP WORD main  ; initialisation de PC ORG 0x10 main: MOV R2, #2222 MOV R4, #4444 ; SWAP R2, R4 B main
