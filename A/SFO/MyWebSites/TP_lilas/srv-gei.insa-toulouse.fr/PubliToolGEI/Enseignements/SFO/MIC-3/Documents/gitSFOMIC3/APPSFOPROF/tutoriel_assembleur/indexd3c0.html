Utilisation de l'assembleur

Pour lancer l'exécution de l'assembleur, il suffit de rajouter dans la commande lilas, l'option -classToRun=assembleur.MainMIC3. Si votre fichier assembleur exemple.ass se trouve dans le répertoire etape1, vous pourrez l'assembler avec la commande :

$ lilas -classToRun=assembleur.MainMIC3 etape1/exemple.ass -listing -out

Les paramètres -listing et -out sont optionnels.

  • -listing permet d'obtenir le listing de la compilation
  • -out permet de créer le fichier correspondant au contenu de la mémoire (le nom de ce fichier est celui du fichier assembleur avec pour extension ".mem"). En utilisant -out et l'option -listing on obtient également le listing dans un fichier dont le nom est celui du fichier assembleur avec l'extension ".lst".

Vous pouvez également utiliser indirectement l'assembleur via le bouton "load" associé aux mémoires de la carte ROM ou de la carte RAM. Il suffit de sélectionner un fichier dont le nom se termine par ".ass" au lieu de ".mem" pour assembler le programme assembleur avant son chargement dans la mémoire (si aucune erreur n'a été détectée durant l'assemblage).
