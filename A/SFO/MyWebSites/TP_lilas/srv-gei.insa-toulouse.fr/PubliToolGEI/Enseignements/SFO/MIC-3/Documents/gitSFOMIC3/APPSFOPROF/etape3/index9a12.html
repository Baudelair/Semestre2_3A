Programmation du GPIO

Pour programmer une application GPIO, utilisez le module etape3/MainSimulateur.lilas avec la commande :

$ lilas /arch/processeurComplet/ProcesseurComplet2015.lilas -awt=awtEtape3 -script=initEtape3 -test

Vous devez programmer l'algorithme suivant défini dans la syntaxe ada :

procedure testGPIO is begin CR := 1; -- GPIO_0 en sortie, GPIO_7654321 en entrée while true loop ODR := SWITCH; Afficheur7Segments := IDR; end loop; end clignotant;

Après test sur simulateur, vous pourrez utiliser votre circuit sur la carte Nexys3 en utilisant la commande :

$ lilas /arch/processeurComplet/ProcesseurComplet2015.lilas -awt=debugAWT -script=initEtape3 -remote
Attention, si nécessaire, cette commande peut impliquer une compilation Xilinx du circuit, ce qui peut prendre un certain temps (environ 4 minutes).
