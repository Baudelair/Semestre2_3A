- 2 -

Utilisation des interruptions sur la carte Audio

Dans la carte audio, le signal PrêtSPI indiquant qu'une donnée est disponible sur le module SPI est en fait relié à la demande d'interruption numéro 4.

Travail : Prise en compte d'une interruption

Modifiez votre programme filtre audio pour que l'opération d'acquisition-traitement-sortie se fasse sur interruption. Le programme principal qui s'exécutera sur la machine pourra être alors le suivant :

R0 := 0; répéter écrire R0 à l adresse 0x4004; // pour visualiser R0 sur les afficheurs 7 segments R0 := R0 + 1; fin répéter
